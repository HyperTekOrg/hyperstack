# Arete Stacks

Protocol stacks and the `arete-stacks` SDK package.

## Structure

```
stacks/
├── ore/                     # Stack definition (deployable)
│   ├── Cargo.toml               # Rust crate with #[arete] macro
│   ├── arete.toml          # CLI config (outputs to ../sdk/*)
│   ├── src/spec.rs              # Stack definition
│   └── idl/ore.json             # Anchor IDL
│
└── sdk/                         # Generated SDKs (publishable)
    ├── typescript/              # npm: arete-stacks
    │   ├── package.json
    │   └── src/
    │       ├── index.ts         # Re-exports all stacks
    │       └── ore/         # Generated by `a4 sdk create typescript`
    │
    └── rust/                    # crates.io: arete-stacks
        ├── Cargo.toml
        └── src/
            ├── lib.rs           # Feature-gated exports
            └── ore/         # Generated by `a4 sdk create rust`
```

## Deploy a Stack

```bash
cd ore
cargo build                      # Generates .arete/*.ast.json
a4 up                            # Deploy to Arete
a4 sdk create typescript ore     # Output to ../typescript/src/ore/
a4 sdk create rust ore           # Output to ../rust/src/ore/
```

## Use Published Stacks

```typescript
import { ORE_STREAM_STACK } from '@usearete/stacks/ore';
```

```rust
use arete_stacks::ore::OreStreamStack;
```
