# Hyperstack Stacks

Protocol stacks and the `hyperstack-stacks` SDK package.

## Structure

```
stacks/
├── pumpfun/                     # Stack definition (deployable)
│   ├── Cargo.toml               # Rust crate with #[hyperstack] macro
│   ├── hyperstack.toml          # CLI config (outputs to ../sdk/*)
│   ├── src/spec.rs              # Stack definition
│   └── idl/pump.json            # Anchor IDL
│
└── sdk/                         # Generated SDKs (publishable)
    ├── typescript/              # npm: hyperstack-stacks
    │   ├── package.json
    │   └── src/
    │       ├── index.ts         # Re-exports all stacks
    │       └── pumpfun/         # Generated by `hs sdk create typescript`
    │
    └── rust/                    # crates.io: hyperstack-stacks
        ├── Cargo.toml
        └── src/
            ├── lib.rs           # Feature-gated exports
            └── pumpfun/         # Generated by `hs sdk create rust`
```

## Deploy a Stack

```bash
cd pumpfun
cargo build                      # Generates .hyperstack/*.ast.json
hs up                            # Deploy to Hyperstack
hs sdk create typescript         # Output to ../typescript/src/pumpfun/
hs sdk create rust               # Output to ../rust/src/pumpfun/
```

## Use Published Stacks

```typescript
import { PUMPFUN_STACK } from 'hyperstack-stacks/pumpfun';
```

```rust
use hyperstack_stacks::pumpfun::PumpfunTokenEntity;
```
